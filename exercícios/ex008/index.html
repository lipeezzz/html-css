Principais formatações

Negrito/Destaque

Para usarmos uma frase em negrito tipo essa aqui usamos a tag B(não semântica).

e temos tipo essa que é feita com a tag STRONG(semântica).

Itálico/Ênfase

nesta frase temos um termo em itálico com a tag I(nao semantica).

nesta frase temos um termo em ênfase usando a tag EM(semântica).

Texto marcado

podemos fazer um texto marcado usando a tag MARK tipo aqui agora.

Texto grande e pequeno

estamos criando um texto grande e um texto pequeno.

Texto deletado

podemos marcar DEL para indicar um texto excluido/riscado como este.

Texto inserido

podemos marcar um texto como inserido com a tag INS para indicar que foi adicionado depois tipo esse aqui.

existe também a tag U tipo aqui(não semântica).

Texto sobrescrito

para inserir coisas do tipo X20+3 usamos a tag SUP tipo agora X20+3.

Texto subescrito
